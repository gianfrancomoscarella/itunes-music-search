# 🎶 iTunes Music Search

Aplicación web para buscar canciones y artistas usando la API pública de iTunes. Desarrollada con JavaScript, HTML y CSS como entrega final del curso de JavaScript de Coderhouse.

---

## Funcionalidades

- **Búsqueda en vivo** con debounce — resultados mientras escribís, sin necesidad de confirmar
- **Filtro por género** — combinable con la búsqueda de texto
- **Historial de búsquedas** — chips clickeables con tus últimas 8 búsquedas, persistidos en LocalStorage
- **Preview de canciones** — reproductor de 30 segundos al hacer click en un resultado
- **Lista de favoritos** — guardá canciones, con badge contador y protección contra duplicados
- **Sugerencia aleatoria por género** — recibe una canción random del género que elijas
- **Modo oscuro / claro** — preferencia guardada en LocalStorage
- **Canciones populares por defecto** — la app carga contenido sin necesidad de escribir nada

---

## Tecnologías

- JavaScript (ES6+)
- HTML5 / CSS3
- [SweetAlert2](https://sweetalert2.github.io/)
- [API pública de iTunes](https://developer.apple.com/library/archive/documentation/AudioVideo/Conceptual/iTuneSearchAPI/index.html)
